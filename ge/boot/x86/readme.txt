

/bl 
   Contêm o BL.BIN, que é o stage3 para VHD.
   
/bm
   Contêm o BM.BIN, que é o stage2 para VHD.
   
   
/cdbl 
   Contêm o cdbl.bin, que é o stage3 para cd.
   
   
/cdstage2
   Contêm o stage2.bin, que é o stage2 para o cd.
   
   
------------

/iso
    Contêm o stage1 do cd.
	Contêm uma tentativa de criar um .ISO usando nasm.
	O Makefile está na pasta principal do projeto.
	
/vhd
   Contêm o stage1 do vhd.
   Contêm uma criação do VHD usando nasm.
   O Makefile está na pasta principal do projeto.
